## Overview
A clean and animated portfolio landing page built with HTML and CSS. Inspired by a Figma template and recreated from scratch.

## Features
- Font animations on hover
- Text masking with images
- Custom layout and CSS transitions

## Inspiration
Based on the [Bentolio Portfolio Template on Figma](https://www.figma.com/design/N3lMnVHAzpNth2KmWQ86KI/Bentolio-%7C-Portfolio-Design-Template--Community-?node-id=0-1)
